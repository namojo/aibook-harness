# AI와 함께 일하는 시대

삼성디스플레이 임직원을 위한 AI 교양서

## 목차

| 챕터 | 제목 | 주요 내용 |
|------|------|----------|
| 1장 | AI의 여정 — 튜링에서 ChatGPT까지 | AI 70년 역사, 겨울과 봄, 딥러닝 혁명 |
| 2장 | 생성형 AI, 왜 지금 이렇게 중요한가 | 기존 AI vs 생성형 AI, LLM 원리, 일하는 방식의 변화 |
| 3장 | 산업별 AI 활용 — 이미 일어나고 있는 변화 | 의료, 금융, 유통, 미디어, 농업/에너지 사례 |
| 4장 | 제조 AI — 스마트 팩토리의 핵심 엔진 | 예지 보전, 비전 검사, 공정 최적화, 디지털 트윈 |
| 5장 | 2024~2026 AI 트렌드 | 멀티모달, AI 에이전트, SLM, 온디바이스, 거버넌스 |
| 6장 | 생성형 AI 시대, 우리는 어떻게 준비할 것인가 | AI 리터러시, 프롬프트 엔지니어링, 협업 마인드셋 |

## 제작 과정

이 도서는 Claude Code의 에이전트 팀 하네스를 활용하여 집필되었습니다.

- **AI Researcher** — 최신 AI 트렌드 및 사례 웹 리서치
- **Book Writer A** — 1~3장 집필
- **Book Writer B** — 4~6장 집필
- **Book Editor** — 전체 원고 검수 및 최종 편집

## 파일 구조

```
_workspace/
  01_researcher_data.md    # 리서치 자료
  02_writer-a_chapters.md  # 1~3장 초안
  02_writer-b_chapters.md  # 4~6장 초안
  03_editor_review.md      # 에디터 검수 결과
  04_final_book.md         # 최종 완성본
```
