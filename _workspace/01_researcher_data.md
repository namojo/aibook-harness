## 1. AI 역사의 주요 이정표 (1950~2025)

### 타임라인

| 연도 | 사건 | 의의 |
|------|------|------|
| **1950** | 앨런 튜링, "Computing Machinery and Intelligence" 발표 | 튜링 테스트 제안, AI 개념의 시초 |
| **1956** | 다트머스 회의 (Dartmouth Conference) | 존 매카시가 "인공지능(Artificial Intelligence)" 용어를 최초로 사용 |
| **1966** | ELIZA 챗봇 개발 (MIT, 조셉 와이젠바움) | 최초의 자연어 처리 대화 시스템 |
| **1969** | Shakey 로봇 (SRI International) | 최초의 범용 이동 지능 로봇 |
| **1974~1980** | 제1차 AI 겨울 (First AI Winter) | 과도한 기대 후 연구 자금 대폭 축소 |
| **1980년대** | 전문가 시스템(Expert Systems) 부흥 | 규칙 기반 AI가 산업에 적용, 이후 제2차 AI 겨울 도래 |
| **1997** | IBM Deep Blue, 체스 세계 챔피언 가리 카스파로프 격파 | 초당 2억 개 포지션 평가, AI가 인간 전문가를 이긴 상징적 사건 |
| **2011** | IBM Watson, Jeopardy! 퀴즈쇼 우승 | 자연어 이해와 대규모 데이터 검색 능력 입증 |
| **2012** | AlexNet, ImageNet 대회 압도적 우승 | 딥러닝 혁명의 시작, CNN(합성곱 신경망)의 위력 증명 |
| **2014** | 이안 굿펠로, GAN(생성적 적대 신경망) 발명 | 생성 모델의 혁신, AI 이미지 생성의 토대 |
| **2016** | Google DeepMind AlphaGo, 이세돌 9단 격파 | 바둑에서 AI가 인간 최고수를 이김, 강화학습의 위력 증명 |
| **2017** | "Attention Is All You Need" 논문 발표 (Google) | **트랜스포머(Transformer)** 아키텍처 탄생, 현대 LLM의 기반 |
| **2018** | Google BERT / OpenAI GPT-1 출시 | 사전학습-미세조정 패러다임 확립 |
| **2020.6** | OpenAI GPT-3 출시 (1,750억 파라미터) | 대규모 언어모델 시대 개막, 퓨샷 학습 능력 시연 |
| **2021** | GitHub Copilot 출시 / DALL-E 1 공개 | AI 코드 어시스턴트 및 텍스트→이미지 생성 상용화 시작 |
| **2022.11** | **ChatGPT(GPT-3.5) 출시** | **5일 만에 100만 사용자, 2개월 만에 1억 사용자** 돌파. 생성형 AI 대중화의 분수령 |
| **2023.3** | GPT-4 출시 (멀티모달) | 텍스트+이미지 입력 처리, 전문가 수준의 추론 능력 |
| **2023.12** | Google Gemini 출시 | 네이티브 멀티모달 모델, Google의 LLM 경쟁 본격화 |
| **2024.3** | Anthropic Claude 3 Opus 출시 | GPT-4에 필적하는 추론 능력, 200K 컨텍스트 윈도우 |
| **2024.5** | **OpenAI GPT-4o 출시** | 음성·비전·텍스트를 실시간 통합 추론하는 옴니 모델 |
| **2024.7** | Meta Llama 3.1 405B 출시 | 역대 최대 오픈소스 LLM, GPT-4급 벤치마크 성능 |
| **2024.9** | OpenAI o1-preview 출시 | 체인오브솟(Chain-of-Thought) 추론에 특화된 최초의 "추론 모델" |
| **2025.1** | **DeepSeek R1 출시** | 오픈소스 추론 모델, 미국 앱스토어 1위 달성. Nvidia 시가총액 하루 만에 약 6,000억 달러 하락 유발 |
| **2025** | Claude Opus 4.5, GPT-5, AI 에이전트 상용화 | 에이전트 AI 시대 개막, 자율적 계획·실행·적응 가능한 AI 시스템 등장 |

### 5-2. AI 에이전트 (Agentic AI)

- 2024~2025년의 **가장 변혁적 트렌드**: 프롬프트 응답을 넘어 자율적으로 계획·실행·적응하는 AI

- McKinsey 2025 보고서: 기업의 **23%가 에이전틱 AI 시스템을 스케일링** 중, **39%가 실험 중**

### 5-3. SLM — 소형 언어 모델 (Small Language Models)

- **가장 놀라운 변화** 중 하나로 부상
- Microsoft **Phi-3-mini**: 단 **38억 파라미터**로 Google PaLM(5,400억 파라미터)과 동일한 MMLU 성능 달성 → 모델 크기 **142배 축소**
- 엣지 디바이스 배포, 비용 효율, 특정 도메인 맞춤화에 최적
- 데이터셋·벤치마크 분야의 연구가 폭발적으로 증가

### 5-4. 온디바이스 AI (On-Device AI / Edge AI)

- 2026년 핵심 트렌드: 데이터를 생성 위치에서 직접 수집·처리·분석
- 실시간 성능, 최소 네트워크 의존성, 낮은 지연시간 제공
- 스마트폰 NPU, IoT 기기, 자율주행 차량 등에 적용 확대
- Apple Intelligence, Samsung Galaxy AI 등 모바일 온디바이스 AI 경쟁 본격화
